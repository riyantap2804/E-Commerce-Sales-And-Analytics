# E-Commerce-Sales-Analytics (Interactive Dashboard Creation Using MS-Excel)
## Project Objective
The Riyan Store aims to create an Annual Sales Report for 2025 to understand customer purchasing behavior, sales performance and key business trends.
The insights from this analysis help the store identify growth opportunities and make data-driven decisions to improve sales performance in upcoming years.

# E-Commerce-Sales-Analytics (Interactive Dashboard Creation Using MS-Excel)
## Project Objective
The Riyan Store aims to create an Annual Sales Report for 2025 to understand customer purchasing behavior, sales performance and key business trends.
The insights from this analysis help the store identify growth opportunities and make data-driven decisions to improve sales performance in upcoming years.

## Dataset used
### Dataset Access Link (<a href="https://github.com/riyantap2804/E-Commerce-Sales-And-Analytics/blob/main/E-Commerce%20Sales%20Analytics(Project).xlsx">Riyan Store</a>)

<p>The dataset contains 2025 e-commerce transactional data including:</p>
<ul>
  <li>Order details and sales values</li>
  <li>Customer demographics (Age Group, Gender)</li>
  <li>Product categories</li>
  <li>Sales channels</li>
  <li>Order status</li>
  <li>State-wise performance</li>
</ul>
